Тулбокс seismo.tbx

Справка к тулбоксу версии 2012-10-18

Тулбокс содержит инструмент «density», позволяющий подсчитать значение плотности сейсмоизученности для указанного участка.

Входной параметр один, по имени «inputPolygon», представляет собой полигон, внутри которого нужно узнать значение сейсмоплотности. Тип параметра «FeatureSet». Несмотря на то, что данный тип позволяет задать на входе много полигонов, обработан будет только один, первый.

При задании полигона следует обеспечить выполнение требований ArcGIS к формату записи: координаты точек полигона (направление рисования) идут по часовой стрелке, нет самопересечений. Для автоматического исправления возможных ошибок ввода вы можете использовать службу ArcGIS http://tasks.arcgisonline.com/ArcGIS/rest/services/Geometry/GeometryServer/simplify

Выходных параметров три, в порядке следования:
seismoDens — единицы измерения km/km2
profilesLength — единицы измерения  km
shapeArea — единицы измерения km2
Тип параметров «double», значение по умолчанию «-1.0».

Известные ограничения связаны с используемыми системами координат и заключаются в следующем.
Возможны сбои при вычислении на территории в зоне меридиана 180 градусов.
Вычисление сейсмоплотности для огромных территорий может также вызвать сбой.

При работе инструмента используется фичекласс с данными сейсмопрофилей. Если доступа к этим данным нет, вычисления невозможны.

Если используется версия инструмента взаимодействующая с БД Oracle, необходимо обеспечить доступность используемых в этой БД данных и хранимой функции, выполняющей вычисления.
